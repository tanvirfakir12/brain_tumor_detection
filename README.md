# 🧠 Brain Tumor Detection App

A deep learning web application that detects brain tumors from MRI scan images using CNN and VGG16 transfer learning.

## 🔍 What it does
- Upload any brain MRI image
- Instantly detects if tumor is present or not
- Shows confidence percentage
- Warns if uploaded image is not a valid MRI scan

## 🛠️ Tech Stack
- Python
- TensorFlow / Keras
- VGG16 Transfer Learning
- Streamlit
- - OpenCV

## 📊 Model Performance
- Algorithm: VGG16 Transfer Learning
- Dataset: Brain MRI Images (Kaggle)
- Training Images: 253 MRI scans
- Accuracy: ~88%

## 🚀 How to Run Locally
pip install -r requirements.txt
streamlit run app.py
